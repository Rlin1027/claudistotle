# Academic Writing Standards

## Paper Structure Templates

### Analytic Philosophy Paper

```
1. Introduction (10-15%)
   - Opening hook: a puzzle, counterexample, or surprising claim
   - Thesis statement (clear, debatable, specific)
   - Roadmap: "In §1 I argue... In §2 I consider... In §3 I show..."

2. Background / Setup (10-15%)
   - Define key terms precisely
   - State assumptions explicitly
   - Present the target argument/position being addressed

3. Main Argument (30-40%)
   - Present your argument in clearly numbered premises
   - Defend each premise with evidence or sub-arguments
   - Use thought experiments when helpful
   - Present argument in both informal and semi-formal form if possible

4. Objections & Responses (20-25%)
   - Present the STRONGEST version of each objection (principle of charity)
   - Respond point-by-point
   - Concede what should be conceded
   - Show why your thesis survives the objections

5. Conclusion (5-10%)
   - Restate the thesis and what was shown
   - Acknowledge limitations honestly
   - Suggest implications and further questions
```

### Continental Philosophy / Hermeneutic Paper

```
1. Problem Situation (10-15%)
   - Why does this text/concept/phenomenon demand interpretation?
   - What is at stake in getting this right?
   - Position within the relevant tradition

2. Textual Analysis / Phenomenological Description (25-30%)
   - Close reading of primary sources with extensive quotation
   - Phenomenological description of the experience/phenomenon
   - Etymological analysis when relevant (especially for Heidegger, Derrida)

3. Interpretive Development (25-30%)
   - Build your interpretation layer by layer
   - Show how your reading illuminates what others have missed
   - Engage with the hermeneutic circle explicitly

4. Critical Engagement (15-20%)
   - Engage with alternative interpretations
   - Show how your reading is superior or complementary
   - Situate within broader scholarly debates

5. Implications (10-15%)
   - What does this interpretation open up?
   - How does it transform our understanding?
   - What further questions does it raise?
```

### Comparative Philosophy Paper

```
1. Introduction (10%)
   - State the philosophical problem that motivates comparison
   - Justify the comparison (not just "both traditions discuss X")
   - State your thesis about what the comparison reveals

2. Methodological Framework (10%)
   - Address the dangers of false equivalence
   - Explain your comparative method
   - Acknowledge cultural-linguistic limitations

3. Tradition A Analysis (20-25%)
   - Present the concept/argument in its own terms
   - Use original-language terminology with translations
   - Situate within its intellectual-historical context

4. Tradition B Analysis (20-25%)
   - Same structure as above
   - Resist the urge to constantly reference Tradition A

5. Comparative Analysis (20-25%)
   - Points of genuine convergence (with caveats)
   - Points of productive divergence
   - What each tradition illuminates about the other
   - What the comparison reveals that neither tradition alone can

6. Conclusion (5-10%)
   - What the comparison contributes to understanding the problem
   - Implications for cross-cultural philosophy
```

## Scholarly Prose Style Guide

### Principles

1. **Clarity above all**: If a sentence requires re-reading, rewrite it
2. **One claim per paragraph**: Topic sentence → support → analysis → transition
3. **Active voice preferred**: "Kant argues" not "It is argued by Kant"
4. **Define before using**: Every technical term gets a definition on first use
5. **Qualify honestly, not excessively**: "I argue that X" not "It might perhaps be the case that one could tentatively suggest X"

### Common Problems and Fixes

| Problem | Example | Fix |
|---------|---------|-----|
| **Throat-clearing** | "In order to understand X, it is first necessary to..." | Cut. Start with the substance. |
| **Hedge overdose** | "It seems that perhaps one might argue..." | "I argue..." or "This suggests..." |
| **Passive evasion** | "It has been argued that..." | "Smith (2020) argues..." or "I argue..." |
| **Citation dump** | "(Smith 2019; Jones 2020; Lee 2021; Park 2022)" | Engage with each, or cite only the most relevant |
| **Jargon without definition** | "The apophantic structure of the as-structure..." | Define each term on first use with a brief gloss |
| **Vague reference** | "This is important for several reasons" | Specify: "This distinction matters because..." |

### Transition Strategies

| Relationship | Transitions |
|-------------|------------|
| Building on | "Moreover," "Furthermore," "Building on this," |
| Contrasting | "However," "By contrast," "Nevertheless," |
| Conceding | "Admittedly," "To be sure," "One might object that" |
| Exemplifying | "For instance," "Consider the case of," |
| Concluding | "Therefore," "It follows that," "This shows that" |
| Turning | "Having established X, I now turn to Y" |

## Differences Between English and Chinese Academic Writing

| Aspect | English Academic Writing | Chinese Academic Writing |
|--------|--------------------------|--------------------------|
| Thesis Statement | Thesis stated directly in opening paragraph | May first establish background, then develop thesis gradually |
| Citation Style | In-text embedded citations predominant | Footnote citations more common |
| Rhetorical Tone | Direct statement ("I argue") | Often uses more subtle expression ("This writer believes," "In my view") |
| Paragraph Structure | One topic per paragraph, strictly enforced | Paragraphs can be longer, containing multiple sub-points |
| Terminology Handling | Use original terms, define on first appearance | Translated term + original term in parallel (e.g., Dasein) |
